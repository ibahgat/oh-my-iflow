为主流 IDE 定制的插件

无缝集成到你熟悉的开发环境中，提升开发效率

1

打开 VSCode 扩展市场

在 VSCode 中按下 Ctrl+Shift+X 打开扩展面板

2

搜索 iFlow 插件

在搜索框中输入 "iFlow" 并找到官方插件

3

点击安装

点击安装按钮，等待插件下载并安装完成

4

重启 VSCode

重启 VSCode 使插件生效，开始使用

1

打开插件设置

2

搜索 Marketplace

在 Marketplace 中搜索 "iFlow" 插件

3

安装插件

点击 Install 按钮安装插件

4

重启 IDE

重启 IDE 使插件生效
